## Development

**Run the dev server:**

```bash
bun run dev
```

**Lint all the files:**

```bash
bun run lint
```

**Format all files with Prettier:**

```bash
bun run format
```

**Check TypeScript issues**

```bash
bun run type-check
```

## Deployment

First, build your app for production:

```sh
bun run build
```

Then run the app in production mode:

```sh
bun start
```
