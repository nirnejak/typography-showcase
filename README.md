## Development

**Run the dev server:**

```bash
npm run dev
```

**Lint all the files:**

```bash
npm run lint
```

**Format all files with Prettier:**

```bash
npm run format
```

**Check TypeScript issues**

```bash
npm run type-check
```

## Deployment

First, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```
